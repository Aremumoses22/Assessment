## ğŸ“ If They Ask Technical Questions

### "Why Redux Toolkit instead of Context API?"
"Redux Toolkit provides better scalability, middleware support, DevTools integration, and is the industry standard for complex state management. While Context API works for simple cases, Redux Toolkit offers better performance with memoization and is easier to debug with time-travel debugging."

### "Why Expo instead of React Native CLI?"
"Expo provides a faster development cycle with managed builds, easy OTA updates, and built-in libraries. For this case study and MVP development, Expo offers the best balance of speed and functionality. However, I'm also comfortable with bare React Native CLI when native modules are required."

### "How did you handle offline storage?"
"I integrated AsyncStorage with Redux Toolkit's async thunks. The app automatically persists the entire product state to local storage whenever it changes, and loads it back when the app starts. This provides a seamless offline experience."

### "How would you scale this app?"
"I'd add: (1) Cloud sync with Firebase, (2) User authentication, (3) Product categories and search, (4) Multiple images per product, (5) Unit and E2E tests, (6) CI/CD pipeline, (7) Analytics and crash reporting with Sentry."
